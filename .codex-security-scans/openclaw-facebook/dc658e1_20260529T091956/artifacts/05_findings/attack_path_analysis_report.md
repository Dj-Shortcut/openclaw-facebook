# Attack-Path Analysis Summary

`CAND-002` is high severity because signed WhatsApp deliveries can be replayed into state-changing and cost-bearing handlers without the replay control already used for Facebook events.

`CAND-001`, `CAND-003`, `CAND-004`, and `CAND-005` are medium severity. No critical unauthenticated RCE, direct secret disclosure, SQL injection, cross-user tRPC data exposure, or Messenger source-image SSRF was validated in reviewed shards.
